AI Playground

AI Playground is a personal collection of original browser games and interactive web experiments.

Purpose

The site is built to explore small, playable ideas in the browser. Each project focuses on a different interaction pattern, such as grid movement, card matching, block placement, live cellular materials, rhythm timing, or tactical planning.

The home page links only to projects that have a real playable screen. Placeholder or unfinished pages are not used as destinations for visitors or for advertising.

Content

The games, interface copy, rules, scoring systems, and update notes are original site content created for this playground. Some projects save scores or preferences so that visitors can compare runs and continue experimenting.
