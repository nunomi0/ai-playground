Original browser games and small interactive experiments, built as self-contained web projects with playable mechanics, visible controls, and notes that explain what each project does.

What is here

This site collects lightweight games, simulations, and interface prototypes that run in the browser. Every listed project is a playable page with its own interaction: you can steer the snake, mine blocks, match cards, draw a cat for arcade rounds, paint falling sand, or try the strategy and rhythm experiments.

The pages are made for quick play and inspection. Each experiment uses its own controls, scoring, visual style, and interaction rules, so the collection works like a small lab of finished prototypes instead of a single landing page with empty entries.

Project notes

The current collection includes classic arcade play, card matching, block sandbox movement, cellular simulation, rhythm farming, and deterministic tactics. New work is linked only after it has a playable screen and enough context for visitors to understand the interaction.

Monetization should appear only beside substantial publisher content and never on empty, unfinished, error, or navigation-only screens.
